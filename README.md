# 🔬 Science Learning Web App

A modern interactive web application designed to help students learn science through quizzes, experiments, and gamified learning.

---

## 🚀 Features

- 🧪 Virtual Lab Experiments  
- ❓ Interactive Quizzes  
- 🧠 Flashcards for Revision  
- 🏆 Leaderboard System  
- 🔥 Daily Streak & Rewards  
- 💬 Ask Expert / Chat Feature  

---

## 🛠️ Built With

- React (Vite)
- Tailwind CSS
- JavaScript (ES6)
- HTML5 & CSS3
